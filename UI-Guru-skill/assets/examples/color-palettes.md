# 配色方案示例

这些示例展示不同产品类型的配色方案，供参考而非直接使用。

---

## 科技产品配色

```css
/* 主题色 */
--color-primary: #3B82F6;           /* 蓝色 */
--color-primary-dark: #2563EB;      /* 深蓝 */
--color-primary-light: #60A5FA;     /* 浅蓝 */

/* 背景色 */
--color-background: #F9FAFB;        /* 浅灰 */
--color-surface: #FFFFFF;           /* 白色 */

/* 文字色 */
--color-text-primary: #1F2937;      /* 深灰 */
--color-text-secondary: #6B7280;    /* 中灰 */
--color-text-tertiary: #9CA3AF;     /* 浅灰 */

/* 功能色 */
--color-success: #10B981;           /* 绿色 */
--color-warning: #F59E0B;           /* 橙色 */
--color-error: #EF4444;             /* 红色 */
--color-info: #3B82F6;              /* 蓝色 */
```

**适用场景**: SaaS 产品、开发工具、企业应用

## 配色组合建议

### 单色配色
选择一个色相，变化饱和度和明度：
```
主色: #3B82F6 (蓝色)
辅助: #93C5FD (浅蓝)
强调: #1E40AF (深蓝)
```

### 类比配色
选择色环上相邻的颜色：
```
主色: #3B82F6 (蓝色)
辅助: #06B6D4 (青色)
```

### 互补配色
选择色环上相对的颜色：
```
主色: #3B82F6 (蓝色)
强调: #F97316 (橙色)
```

### 三分配色
选择色环上均匀分布的三种颜色：
```
主色: #3B82F6 (蓝色)
辅助: #10B981 (绿色)
强调: #F59E0B (黄色)
```

*注意: 这些是示例配色，实际项目中应根据品牌定位和用户需求定制。*
